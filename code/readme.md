Directory of code associated with Summer 2021 Sea Star Wasting Disease Challenge experiments with adult _Pycnopodia helianthoides_.

## Network Diagram Generation

**Script:** `generate_network_diagram.py`

Python script that generates network diagrams showing relationships between GO Biological Process terms and UniProt IDs from supplemental file S11-DEGlist_enrichedprocesses_2021-2022.tab.

See [README-network-diagram.md](README-network-diagram.md) for detailed usage instructions.
